# Khay2Holiday
  Looking for a website to help in your vacation planning? 
  Look no further. Khay 2's got you covered.
  Skip the long wait in lines to book reservations at your favourite vacation locations
  Register with us now. Whether you're looking for a relaxing getaway, an adventurous trip or luxury stays, Khay 2 connects you with the best destinations, accomodations and travel experiences.

* Khay 2 Holiday is a web-based vacation and booking platform designed to:
  - Help users explore exciting travel destinations
  - Compare accommodation options
  - View pricing and packages
  - Make secure bookings
  - Get travel inspiration and guides


* Features
  - Mordern and responsive design
  - destination search and filtering
  - Accomodation listings
  - Booking systems and reservation flow
  - Contact and support section
 
* This project is built using HTML, Css and Javascript.
